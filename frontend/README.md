# ASTMS Frontend

Modern React + Tailwind CSS frontend for **ASTMS - Automated Software Testing Management System**.

## Run locally

```bash
npm install
npm start
```

## Features

- Role-based login UI (Admin/Tester/Developer)
- Dashboard with KPI cards and charts
- Test case management with search + create modal
- Test execution workflow with result modal + mock history
- Bug tracking table with assignment and status updates
- Reports & analytics charts
- Reusable components for cards, buttons, modal, and tables
